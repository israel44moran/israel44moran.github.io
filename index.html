PROYECTO 06 EN VIVO

Predicción de ventas con Machine Learning

Modelo Random Forest con shrinkage semanal y rezagos anuales, validado con cross-validation rolling de seis ventanas y prueba pareada estadística. Probado en dos datasets reales: UCI Online Retail II (1M+ transacciones B2B/B2C) y Bread Basket Edimburgo (panadería). Único candidato estadísticamente superior al RF de referencia (p = 0.06).

  • scikit-learn
  • Prophet
  • Pandas
  • Streamlit

PROYECTO 09 · TRILOGÍA 1/3 SQL + GRÁFICAS

Observatorio macroeconómico de México · SQL puro

Sesenta y cinco años de indicadores oficiales del Banco Mundial analizados con quince queries SQL avanzadas sobre DuckDB. Window functions, CTEs encadenadas, agregaciones jerárquicas, pivots, subqueries correlacionadas. Cero pandas — el SQL es el producto. Resultados visualizados con cuatro gráficas que cuentan la historia.

  • SQL
  • DuckDB
  • CTEs
  • Window functions
  • Matplotlib

PROYECTO 10 · TRILOGÍA 2/3 EN PRODUCCIÓN

Pipeline orquestado con GitHub Actions

El Observatorio se mantiene solo: cada lunes a las 12:00 UTC un workflow descarga datos frescos del Banco Mundial, recarga DuckDB, ejecuta las quince queries y regenera gráficas. Auto-commit solo si hay cambios reales; auto-issue si algo falla. Idempotente y auditable.

  • GitHub Actions
  • YAML CI/CD
  • Bash
  • Cron
  • github-script
