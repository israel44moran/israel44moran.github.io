# israel44moran.github.io

Portafolio personal de Israel Moran Rivera publicado en GitHub Pages.

URL: https://israel44moran.github.io

## Stack

Sitio estático construido con HTML, CSS y un poco de JavaScript vanilla. Sin frameworks, sin build step.

```
.
├── index.html     # Estructura semántica + contenido
├── styles.css     # Design system + responsive
├── main.js        # Scroll reveals y smooth-scroll
└── README.md
```

## Desarrollo local

```bash
python -m http.server 8000
# Abrir http://localhost:8000
```

## Publicación

Cualquier push a `main` activa el deploy automático de GitHub Pages.
